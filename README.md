# x64 ZoneTool

## Supported Games
* **IW6** (*Call of Duty: Ghosts*)
* **S1** (*Call of Duty: Advanced Warfare*)
* **H1** (*Call of Duty: Modern Warfare Remastered*)
* **H2** (*Call of Duty: Modern Warfare 2 Campaign Remastered*)

## Commands
* `loadzone <zone>`
* `unloadzones`
* `verifyzones <zone>`
* `dumpzone [target game] <zone>`
* `dumpasset <type> <name>`
